音频修补
  1. 安装最新的 Lilu + AppleALC
  2. 通过 Clover Configurator 设置 config.plist/Devices/Audio/Inject=NO
  3. Clover->DSDT 重命名 (如果检测到)
    • AZAL->HDEF
    • HDAS -> HDEF
  4. 在 Hackintool 工具中点击 声音
  5. 通过下拉列表设置 Layout ID
  6. 文件->导出->Clover config.plist (覆盖现有的config.plist文件)
  7.  重新启动
  8. 如果没有音频，请返回步骤4并尝试不同的 Layout ID

注意：如果您尝试所有 Layout ID 但仍有问题，请尝试勾选 核显-> 应用补丁-> 高级-> 仿冒声卡ID 选项。 如果这依然不起作用，那么尝试安装 FakePCIID.kext + FakePCIID_Intel_HDMI_Audio.kext
