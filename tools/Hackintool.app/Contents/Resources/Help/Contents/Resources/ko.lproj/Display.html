디스플레이 EDID 패치

EDID 패치를 사용하면 디스플레이 모델에 맞게끔 해상도를 추가하여 디스플레이 성능을 확장할 수 있습니다

  1. 디스플레이 목록에서 편집할 디스플레이를 선택해 주세요
  2. EDID에 표시되지 않은 해상도의 경우 추가 해상도 범위 입력을 위해 “추가” 버튼을 눌러주세요
  3. 가로 / 세로 픽셀을 입력하여 해상도를 추가하거나 선택한 해상도를 삭제할 수 있습니다
  4. 내보내기 아이콘을 클릭하여 바탕화면에 생성된 파일을 이용해 교정 파일을 생성합니다
  5. 한국 유저는 식스플로우에서 보다 상세한 설명을 확인할 수 있습니다

추출된 EDID 설치

EDID 패치를 설치하기 위해선 디스플레이 재정의와 라이브러리 익스텐션을 사용할 수 있습니다

디스플레이 재정의(DISPLAY OVERRIDES)

  • DisplayVendorID-X 폴더를 /System/Library/Displays/Contents/Resources/Overrides로 복사합니다
  • (엘 캐피탄의 경우/System/Library/Displays/Overrides의 경로)
  • Icon.plist를 /System/Library/Displays/Contents/Resources/Overrides로 복사합니다
  • (엘 캐피탄의 경우 /System/Library/Displays/Overrides의 경로)

라이브러리 익스텐션 (LIBRARY EXTENSION)

  • DisplayEDID-X-X.kext를 /Library/Extensions 또는 EFI/Clover/kexts/Other에 복사합니다
  • (구성에 따라 다르지만, 한국의 경우 주로 EFI/Clover/Kexts/Other에 넣는 것이 트렌드입니다)

시스템 무결성 보호(SIP) 

  1. 상단에서 설명된 경로에 파일을 복사하기 위해선 SIP를 비활성화 해야 합니다
  2. SIP 해제를 위해 클로버 클로버 컨피겨레이터를 실행한 다음 Config.plist/Rt Variables 항목에서
  3. BooterConfig : 0x28, CsrActiveConfig : 0x67을 입력합니다

추가정보

만약 디스플레이 비율이 16:10이고 크기 조정된 해상도를 사용하려면 다음과 같이 EDID 정보를 사용합니다

  • iMac Display
  • MacBook Pro Display
  • Cinema HD Display
  • LED Cinema Display

16:9 비율의 디스플레이를 사용하면서 확장해상도를 사용해야 할 경우 다음과 같이 EDID 정보를 사용합니다

  • Apple Thunderbolt Display (내장 그래픽에는 절대 사용하지 말 것)
  • iMac Retina Display
  • MacBook Air Display
