오디오 레이아웃 ID 패치
  1. 최신 Lilu와 AppleALC를 설치합니다 (EFI/Clover/Kexts/Other)
  2. Confi.plist/Devices/Audio : Inject=NO를 설정
  3. Config.plist/ACPI : HDAS 또는 CVAS를 HDEF로 설정
  4. >커스텀맥 툴에서 Audio 탭 선택
  5. >레이아웃 아이디 목록에서 오디오 레이아웃 아이디 선택
  6. >파일 → 내보내기 → 클로버용 Config.plist (기존 Config.plist에 덮어 씌워짐)
  7. 재부팅
  8. >만약 오디오가 활성화되지 않은다면 다른 레이아웃 아이디로 4번 항목부터 다시 시작

NOTE : 만약 모든 레이아웃 ID를 사용했음에도 여전히 문제가 있을 경우
패치 → 고급 → 오디오 장치 ID 스푸핑 옵션을 설정해 보십시오.
그래도 작동하지 않는다면 FakePCIID.kext, FakePCIID_Intel_HDMI_Audio.kext를
설치하거나 VoodooHDA.kext를 설치하십시오
