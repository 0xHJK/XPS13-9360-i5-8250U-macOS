Display EDID Patching

With EDID patching you are able to apply display model overrides and add scaled resolutions.

  1. From the Displays list select the display you want to edit
  2.  Optionally add / fix the monitor ranges (if your EDID doesn't have them or they are bad)
  3.  Optionally add scaled resolutions by clicking “Add” and then clicking on the values to edit them. You can remove entered resolutions by the “Delete” button after selecting them.
  4.  Click the “Export” button to generates the overrides on the desktop
Installation

There are two methods to install EDID patches; Display Overrides and Library Extensions:

Display Overrides:
  • Copy DisplayVendorID-x folders into /System/Library/Displays/Contents/Resources/Overrides (or /System/Library/Displays/Overrides for < El Capitan)
  • Copy Icons.plist to /System/Library/Displays/Contents/Resources/Overrides (or /System/Library/Displays/Overrides for < El Capitan)
Library Extensions:
  • Copy DisplayEDID-x-x.kext into /Library/Extensions or EFI/CLOVER/kexts/Other (depending on your configuration)
System Integrety Protection (SIP)

You will need to disable SIP to be able to copy the files to the above locations.

Extra Info

If you have a 16:10 display and want to use scaled resolutions you can use:

  • iMac Display
  • MacBook Pro Display
  • Cinema HD Display
  • LED Cinema Display

For 16:9 displays and scaled resolutions you can use:

  • Apple Thunderbolt Display (Do not use for internal display!)
  • iMac Retina Display
  • MacBook Air Display
